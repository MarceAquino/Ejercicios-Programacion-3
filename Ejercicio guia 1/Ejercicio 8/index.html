HTML
Lenguaje de marcado utilizado para estructurar contenido en la web.
CSS
Lenguaje utilizado para definir el estilo y diseño de las páginas web, como colores y fuentes.
JavaScript
Lenguaje de programación utilizado para crear interactividad y dinamismo en páginas web.
API
Interfaz de programación de aplicaciones que permite que diferentes sistemas se comuniquen entre sí.
UX/UI
Diseño de experiencia de usuario (UX) y diseño de interfaz de usuario (UI), enfocados en la mejora de la interacción del usuario con una aplicación o página web.
